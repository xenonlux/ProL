🫀 القلب الناطق – لغة البرمجة العربية الشاملة

القلب الناطق هي بيئة برمجة متكاملة تُمكّن المطور العربي من كتابة الأكواد بلغته الأم، باستخدام مترجم/مفسر متعدد الطبقات يدعم من الأوامر البسيطة إلى البرمجة الكائنية، والشبكات، وقواعد البيانات، والواجهات الرسومية، وحتى توليد كود آلة حقيقي.

المشروع مفتوح المصدر بالكامل، ويتضمن عدة إصدارات (Python، C، WebAssembly/JS، وAssembly) لتغطية مختلف الاحتياجات والبيئات.

---

📌 الميزات الرئيسية

· لغة عربية حرة – يمكنك كتابة الأوامر بالعربية بشكل طبيعي (مثل: اعرض "مرحباً"، درب النموذج، اذا $س > 10).
· مترجم / مفسر متعدد المستويات – يدعم:
  · التحليل المعجمي والنحوي العربي المتقدم.
  · آلة افتراضية (VM) خاصة بتعليمات مخصصة.
  · توليد كود آلة x86‑64 (ملفات ELF قابلة للتنفيذ).
· مكتبات مدمجة – رياضيات، رسم بياني، شبكات (HTTP)، قواعد بيانات (SQLite)، واجهات رسومية (Tkinter)، تعدد المهام (خيوط).
· ذاكرة سياق وجلسات – تذكُّر آخر الأهداف والمصادر والأعداد، وحفظ/تحميل الجلسات.
· تصحيح تلقائي – يقترح تصويبات للأخطاء الإملائية الشائعة.
· بيئات متعددة:
  · واجهة أوامر تفاعلية (REPL) بـ Python / C.
  · IDE ويب كامل (qalb‑ide.html) يعمل في المتصفح.
  · مترجم مستقل بلغة C.
  · آلة افتراضية مكتوبة بـ Assembly.

---

🗂️ هيكل المشروع

الملف الوصف
قلب.py إصدار أساسي: محلل عربي مع ذاكرة سياق وتوليد تعليمات VM بسيطة.
قلب_كامل.py إصدار شامل: دوال، شروط، حلقات، مكتبات، ملفات، رسم، ذكاء اصطناعي بسيط، تعدد مهام، VM متكاملة.
قلب_متقدم.py إصدار متقدم: يضيف OOP (فئات، وراثة، كائنات)، استثناءات، شبكات HTTP، قواعد بيانات SQLite، واجهة رسومية (tkinter).
قلب_متكامل.py إصدار سريع: VM خفيفة، محلل عربي بأوامر أساسية، مناسب للأجهزة المحدودة.
قلب_نهائي.py إصدار نهائي: مترجم عربي ← VM ← كود آلة x86‑64 (ELF حقيقي).
قلب.c مترجم/مفسر بلغة C: يعمل بدون Python، يدخل تفاعلياً أو من ملف، ينتج ملفات ELF.
vm_heart.asm آلة افتراضية مكتوبة بلغة Assembly x86‑64 (تنسيق ELF).
qalb-ide.html IDE ويب كامل (JavaScript) – محرر مع تمييز نحوي، أمثلة جاهزة، مفسر عربي متكامل، وعرض شجرة البناء (AST).

⚙️ متطلبات التشغيل

🔹 الإصدارات (Python)

· Python 3.6+.
· المكتبات الإضافية حسب الميزات (اختيارية):
  · numpy – للمصفوفات والتدريب.
  · matplotlib – للرسم البياني.
  · sqlite3 (مضمنة) – لقواعد البيانات.
  · tkinter – للواجهات الرسومية.

يمكن تثبيت المكتبات المطلوبة (إن وجدت) باستخدام:

```bash
pip install numpy matplotlib
```

🔹 الإصدار (C)

· gcc أو أي مترجم C يدعم -m64.
· لا يحتاج إلى مكتبات خارجية.

🔹 IDE الويب

· أي متصفح حديث (يدعم JavaScript ES6).
· لا يحتاج إلى خادم، فقط افتح qalb-ide.html مباشرة.

🚀 كيفية الاستخدام

1. وضع التفاعل المباشر (REPL) – Python

```bash
python قلب_كامل.py
```

ثم اكتب جملة عربية، مثل:

```
📝 > اعرض "مرحباً بالعالم"
```

سيُظهر المحلل فهمه ثم يُطلب التأكيد، وبعدها يُنفذ.

2. تشغيل ملف برنامج عربي

```bash
python قلب_كامل.py برنامجي.قلب
```

حيث برنامجي.قلب ملف نصي يحتوي على أوامر عربية (كل جملة تنتهي بنقطة .).

3. استخدام المترجم بلغة C

```bash
gcc قلب.c -o قلب
./قلب
```

أو لتشغيل ملف:

```bash
./قلب برنامجي.قلب
```

4. استخدام IDE الويب

افتح qalb-ide.html بأي متصفح. ستجد:

· محرراً يميناً-يساراً مع تمييز نحوي.
· أمثلة جاهزة بالنقر على اليسار.
· زر تشغيل الكود لتنفيذ ما كتبته مباشرة.
· لوحة المخرجات وشجرة البناء (AST).

📝 أمثلة برمجية

مثال 1 – تعريف متغير وعرضه

```
$الاسم = "أحمد".
اعرض "مرحباً " + $الاسم.
```

مثال 2 – الشرط (اذا)

```
$الدرجة = 85.
اذا $الدرجة >= 90:
    اعرض "ممتاز".
والا اذا $الدرجة >= 70:
    اعرض "جيد".
والا:
    اعرض "راسب".
نهاية
```

مثال 3 – حلقة طالما

```
$عداد = 1.
طالما $عداد <= 5:
    اعرض "العدد: " + $عداد.
    $عداد = $عداد + 1.
نهاية
```

مثال 4 – دالة (إصدارات متقدمة)

```
دالة مربع(عدد):
    ارجع عدد * عدد.
نهاية

$نتيجة = مربع(7).
اعرض $نتيجة.
```

مثال 5 – برمجة كائنية (في قلب_متقدم.py أو IDE الويب)

```
صنف حيوان:
    $اسم = "حيوان".
    دالة تكلم():
        اعرض "أنا " + $اسم.
    نهاية
نهاية

$ك = جديد(حيوان).
$ك.تكلم().
```

🧠 الأوامر الرئيسية المدعومة

أوامر أساسية

العربية المعنى
اعرض طباعة قيمة
اذا ... والا ... نهاية شرط
طالما ... نهاية حلقة شرطية
لكل ... في ... نهاية حلقة تكرارية
دالة ... نهاية تعريف دالة
ارجع إرجاع قيمة من دالة

أوامر متقدمة (حسب الإصدار)

· صنف – تعريف فئة.
· جديد – إنشاء كائن.
· حاول ... امسك – معالجة الاستثناءات.
· استورد – استيراد مكتبة (رياضيات، رسم، شبكات، إلخ).
· اقرأ / اكتب – التعامل مع الملفات.
· نفذ_بالتوازي – خيوط متعددة.
· احصل / ارسل – طلبات HTTP.
· قاعدة_بيانات / استعلام_قاعدة – استعلامات SQLite.
· نافذة / زر / نص – إنشاء واجهات رسومية (tkinter).

لمشاهدة القائمة الكاملة في أي إصدار، اكتب الأمر مساعدة داخل البيئة التفاعلية.

---

⚙️ تفاصيل الآلة الافتراضية (VM)

يحتوي المشروع على VM خاصة بتعليمات مثل:

· LOAD – تحميل قيمة.
· ADD, SUB, MUL, DIV – العمليات الحسابية.
· JMP, JZ, JNZ – القفز المشروط.
· PRINT – طباعة السجل.
· HALT – إيقاف التنفيذ.

في الإصدارات المتقدمة، يتم تحويل الأوامر العربية إلى هذه التعليمات ثم تنفيذها، أو توليد كود آلة x86‑64 حقيقي (ELF).
